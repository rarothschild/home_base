Project Case Study

Base Defender

GitHub

Fly the ship to ram the enemies or block their way with the boxes. Don't let them reach your base!

Built in C++, this game utilizes Emscripten to convert the code into html usable on my site.

Controls:

  • Arrow Up: thrust upward
  • Arrow Left / Right: move horizontally
  • Space Bar: drop obstacles currently connected to your red rope

Note: This game is currently just a test, press r to spawn enemies. Real levels coming soon!
